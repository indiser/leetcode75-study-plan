## 🚀 Quick Start

### Prerequisites
```bash
# Ensure you have a C++ compiler installed
g++ --version
```

### Running a Solution
```bash
# Navigate to the problem category
cd "Leetcode 75/<Category>/"

# Compile the solution
g++ ProblemName.cpp -o ProblemName

# Execute
./ProblemName        # Linux/Mac
ProblemName.exe      # Windows
```

## 💡 Learning Approach

This repository follows a structured learning path:

1. **Foundation First** - Starting with arrays, strings, and basic techniques
2. **Progressive Complexity** - Moving from simple to advanced data structures
3. **Pattern Recognition** - Identifying common problem-solving patterns
4. **Optimization** - Focusing on time and space complexity improvements
